| route    | description          |
| ------------- | -------------------- |
| /api/message/    | get all messages from room    |

| Event name    | description          |
| ------------- | -------------------- |
| 'addMessage'    | request from FE to add messages to room  |
| 'sendMessage'   | request from BE to send new message to all room users|
